La première rencontre du groupe des utilisatrices et utilisateurs de R de Toulouse est prévue le mardi 25 septembre 2018 de 17h à 18h, à l’amphithéâtre Schwartz de l’Institut de Mathématiques de Toulouse (métro ligne B, Université Paul Sabatier) :

Programme

  • Marylène Henry présentera une initiation à ggplot2 :

Réaliser vos visualisations graphiques avec R – initiation au package ggplot2

ggplot2 est une extension du tidyverse permettant de générer des graphiques avec une syntaxe cohérente et puissante. Cela nécessite l’apprentissage d’une syntaxe de programmation supplémentaire basée sur la grammar of graphics de Wilkinson. L’objectif de cette présentation sera de présenter le langage de construction de graphiques. A partir d’exemples, les différentes couches de création d’un graphique seront présentées. Enfin, il s’agira de donner quelques astuces pour prendre en main rapidement le package et personnaliser ses propres graphiques.

  • Serguei Sokol présentera le package Deriv, qu’il maintient sur CRAN :

Dériver symboliquement des fonctions mathématiques avec R et le package Deriv

The Deriv package offers extended capacities for R code differentiation compared to the classical function deriv() available in R. Deriv can differentiate not only R expressions but whole functions with multiple assignment operators, if-else constructs and so on. The result of a function differentiation is also a function that can be called and/or submitted to optimization routines for calculating gradients and/or hessians. A default table with differentiation rule is quite furnished including classical and special mathematical functions but also probability densities. Moreover, this table can be extended by custom functions provided by user thus adapting differentiation rules to its own situation. We present here basics of ‘Deriv’ functioning and provide few examples illustrating its advantages.
