Aperçu du package Shiny, ou comment construire une application web dynamique directement depuis R-Studio ?, par Marion Aguirrebengoa

Shiny est un package R qui permet de construire facilement une application dynamique et de la rendre accessible dans une webpage ou dans un rapport Rmarkdown, sans aucune connaissance en développement web (html, CSS, javascript, …) en utilisant uniquement R. Grâce à quelques fonctions, Shiny permet de combiner le pouvoir en calcul R avec l’interactivité d’une page web. Cette présentation montrera un petit aperçu de son potentiel.
